Para hacer la conexion con mySQL, crear un archivo ".env" con la configuracion de la base de datos
EJEMPLO:

FLASK_SECRET_KEY=clave_super_secreta_cambia_esto
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=tu_password_real
MYSQL_DB=proyectomod

Algunas de las librerias necesarias:

mysql-connector-python
Werkzeug
Flask
python-dotenv
